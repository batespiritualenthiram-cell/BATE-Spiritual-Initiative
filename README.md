## 🚀 Quick Start

### 1. Prerequisites
- Node.js installed on your system.

### 2. Backend Setup
```bash
cd backend
npm install
node server.js
```

### 3. Accessing the Platform
- Visit `http://localhost:3000` to access the main site.
- **Admin Access**: Click the BATE logo on the home page and enter the administrator password.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (ES6+).
- **Backend**: Node.js, Express.
- **Libraries**: 
    - `Canvas-Confetti`: For high-end success celebrations.
    - `AOS`: For smooth scroll animations.
    - `Font Awesome`: For spiritual and technical iconography.
